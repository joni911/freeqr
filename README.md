# 🎨 Free QR Code Generator

Generate beautiful custom QR codes instantly - **100% Free, No Login Required!**

## ✨ Features

- 🎨 **6 Unique Styles** - Squares, Dots, Rounded, Diamonds, Stars, Striped
- 🌈 **Custom Colors** - Choose any foreground & background color
- 📱 **Instant Preview** - See your QR code before downloading
- 📥 **PNG Download** - High-quality PNG with no watermarks
- 🔒 **No Login** - No signup, no database, no tracking
- ⚡ **Lightning Fast** - Runs entirely in your browser

## 🚀 Live Demo

Try it now: **[https://joni911.github.io/freeqr/](https://joni911.github.io/freeqr/)**

## 📦 Deploy Your Own

### Cloudflare Pages
1. Fork this repository
2. Go to Cloudflare Pages → Create Project → Connect to Git
3. Select your forked repo → Deploy

### Vercel
1. Fork this repository
2. Run `vercel` in the project folder, or drag & drop to Vercel dashboard

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Bootstrap 4
- qrcode-generator (JS library)
- Zero backend required!
